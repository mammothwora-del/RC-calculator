# RC Structure Calculator — คู่มือทำให้เป็นเว็บแอปออนไลน์

แอปนี้เป็น **PWA (Progressive Web App)** — เปิดผ่าน URL ได้จากทุกอุปกรณ์,
ใช้งานออฟไลน์ได้, และติดตั้งลงหน้าจอมือถือ/คอมได้เหมือนแอปจริง

## ไฟล์ในชุดนี้
- `index.html` — ตัวแอปหลัก
- `manifest.json` — ข้อมูลแอป (ชื่อ ไอคอน สี)
- `service-worker.js` — ทำให้ใช้งานออฟไลน์ได้
- `icon-192.png`, `icon-512.png`, `icon-512-maskable.png` — ไอคอนแอป

> สำคัญ: ต้องอัปโหลด **ทุกไฟล์ไว้โฟลเดอร์เดียวกัน** และเปิดผ่าน **https://** (ไม่ใช่เปิดไฟล์ตรงๆ) PWA จึงจะทำงานเต็มรูปแบบ

---

## วิธีที่ 1: Netlify Drop (ง่ายสุด ไม่ต้องใช้ command line) ⭐ แนะนำ

1. ไปที่ https://app.netlify.com/drop
2. ลากโฟลเดอร์ทั้งหมด (หรือไฟล์ ZIP) ไปวางในหน้าเว็บ
3. รอสักครู่ จะได้ URL ทันที เช่น `https://random-name-123.netlify.app`
4. (ตัวเลือก) สมัครบัญชีฟรีเพื่อเปลี่ยนชื่อ URL และเก็บไว้ถาวร

ฟรี ใช้ https อัตโนมัติ PWA ทำงานเต็มรูปแบบ

---

## วิธีที่ 2: GitHub Pages (ฟรี ถาวร เหมาะถ้ามีบัญชี GitHub)

1. สร้าง repository ใหม่บน GitHub (เช่นชื่อ `rc-calc`) ตั้งเป็น Public
2. อัปโหลดไฟล์ทั้งหมดเข้า repo (กดปุ่ม "Add file" → "Upload files")
3. ไปที่ Settings → Pages
4. ใต้ "Branch" เลือก `main` / โฟลเดอร์ `/ (root)` แล้วกด Save
5. รอ 1-2 นาที จะได้ URL: `https://<username>.github.io/rc-calc/`

---

## วิธีที่ 3: Cloudflare Pages (ฟรี เร็ว)

1. ไปที่ https://pages.cloudflare.com
2. เชื่อมกับ GitHub repo หรืออัปโหลดโฟลเดอร์โดยตรง (Direct Upload)
3. ตั้ง build command = (เว้นว่าง), output directory = `/`
4. Deploy จะได้ URL `https://<project>.pages.dev`

---

## การติดตั้งเป็นแอปบนเครื่อง (หลัง deploy แล้ว)

- **Android (Chrome):** เปิด URL → เมนู ⋮ → "ติดตั้งแอป / Add to Home screen"
  หรือกดปุ่ม "⬇ ติดตั้งแอป" สีส้มที่มุมบนของหน้า
- **iPhone (Safari):** เปิด URL → ปุ่มแชร์ → "เพิ่มลงในหน้าจอโฮม"
- **คอม (Chrome/Edge):** ไอคอนติดตั้งในแถบ address bar หรือปุ่ม "⬇ ติดตั้งแอป"

ติดตั้งแล้วเปิดใช้ได้แม้ไม่มีเน็ต (ยกเว้นฟอนต์/แผนที่ที่ต้องโหลดครั้งแรก)

---

## ทดสอบในเครื่องก่อน deploy (ตัวเลือก)
หากมี Python:
```
cd โฟลเดอร์ของแอป
python3 -m http.server 8000
```
แล้วเปิด `http://localhost:8000`

---

## หมายเหตุ
- เปิดไฟล์ `index.html` แบบดับเบิลคลิกตรงๆ ใช้ได้เฉพาะตัวคำนวณ แต่ฟีเจอร์
  ติดตั้งแอป/ออฟไลน์จะไม่ทำงาน (ต้องผ่าน http/https)
- หากแก้ไขโค้ดแล้วอัปเดต ให้เปลี่ยนเลขเวอร์ชันใน `service-worker.js`
  (`rc-calc-v2` → `rc-calc-v3`) เพื่อให้เบราว์เซอร์โหลดเวอร์ชันใหม่
